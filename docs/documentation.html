FrogSYS documentation

Pre-installation/Pre-use

Check that dr(number) doesn't exist. No directories should exist.
If they do, you do not have a clean install. Delete the drives and clear CFG.


Creating disks

Begin by launching diskpart 

# diskpart
The terminal will then ask for a disk number. Name this accordingly. 
# disk number=NUMBER
Give your disk a name. Example "General Files." 
# disk name=General Files
Mount the boot to your choice.

IMPORTANT: Mounting boot incorrectly can cause serious errors and may need config changes. When wanting to mount boot, enter the following

 
# Mount Boot [Y/N]=Y
After finishing diskpart, open the directory eg. /dr1/ and run finishDiskPart.bat to create archives and packages.


PKG

PKG is the pre-installed (clean installation) package wrapper/manager for FrogSYS.
Launch PKG: 

# pkg

Installing packages [.pkg]

Load the installer: 
# install
Pick a drive/disk to install to: 
# drive number=3
Choose a package from /packages/[name].pkg eg hello.pkg 
# package?hello
The package will then install.

Archiving packages [.bat]

PKG Archiving is buggy and shouldn't be used often; Packages like ARCHIVAL should be used for archiving.

 Load the archiver 
# archive
Pick a drive/disk to archive to 
# drive to archive to=NUMBER
Get the directory of the .bat 
# directory=
Give the package a name/new name: 
# package name?hello
